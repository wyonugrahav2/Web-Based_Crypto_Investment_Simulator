# - gunicorn disertakan sebagai web server production-grade untuk deployment
#   di server Linux/VPS. Untuk deployment ke Vercel, gunicorn tidak dipakai
#   karena Vercel menjalankan aplikasi lewat runtime WSGI serverless-nya
#   sendiri (lihat vercel.json), tapi paket ini tetap aman untuk diinstal.
